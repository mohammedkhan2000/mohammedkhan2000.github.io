Algorithm Visualiser

This project involved developing an Algorithm Visualiser that allows users to see the exact process that a sorting algorithm uses to organise a set of data. The visualiser supports multiple sorting algorithms, including Bubble Sort, Merge Sort, and Quick Sort. Users can input their own data sets and watch as the algorithms sort the data step-by-step. The project was built using React for the user interface, TypeScript for type safety, and SASS for styling. This visualiser serves as an educational tool to help users understand the inner workings of sorting algorithms.

  • SASS
  • TypeScript
  • React
  • Sorting Algorithms
  • Data Visualisation
  • User Interface
